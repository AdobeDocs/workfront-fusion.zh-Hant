# 函式概觀

對應項目時，您可以使用函式來建立簡單或複雜的公式。可用的函式與 Excel 和一些程式語言中的函式類似：

* 這些函式會評估一般邏輯、數學、文本、日期和陣列。
* 您可以利用函式來執行條件邏輯和項目值轉換，例如將文字轉換為大寫、修剪文本、轉換不同的日期格式等。
