# Adobe Workfront Fusion 概觀

Adobe Workfront Fusion 的角色是將您的流程自動化，讓您的使用者不需要花費太多時間處理例行性任務。其將應用程式和服務內部及兩者之間的動作連結起來，並建立一個自動轉移和轉換您的資料的情境。您建立的情境會監視應用程式或服務中的資料，並處理該資料以便提供您想要的結果。

Workfront Fusion 中的情境代表特定的工作流程。例如，一個非常簡單的情境可以監視特定 Workfront 請求佇列中的請求，然後使用範本自動將這些請求轉換成專案。

我們可以建立各種情境來處理非常具體的情況和使用案例。您可以使用篩選器並根據資料欄位或屬性來變更動作、疊代處理檔案或資料、將資料儲存至資料存放庫並在執行另一個情境時使用，或以許多其他方式設定您的情境。

您也可以將 Workfront 以外的應用程式新增至情境。Fusion 包含許多適用於外部應用程式的專用連接器，包括由 Adobe、Google、Microsoft 和許多其他廠商提供的應用程式。若某個應用程式沒有專用連接器，Fusion 具備多種方式可利用公開 API 連接至任何服務。

Workfront Fusion 也提供支援情境的功能，例如情境建立範本、執行歷史記錄和偵錯。

>[!NOTE]
>
>Workfront Fusion 僅支援以 Chrome 為基礎的瀏覽器。
