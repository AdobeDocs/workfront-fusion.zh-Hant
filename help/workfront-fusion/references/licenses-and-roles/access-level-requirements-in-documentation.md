# 文件的存取權要求

Adobe WorkfrontFusion文檔how-to文章包含一個表，該表說明了該過程所需的訪問和權限要求。 此「訪問要求」(Access requirements)表允許您瞭解您是否可以在Fusion中執行某個操作，或者您為什麼不能執行該操作。 本文將介紹「訪問要求」表中的每個元素，並提供故障排除提示和指向更深入資訊的連結。

如果給定項目的「訪問要求」表中沒有行，則該操作沒有該類型的要求。

某些行包含標有「新建」和「當前」的資訊。 這是因為Workfront正在向新的定價和包裝模式過渡，一些組織在新模式下運營，其他組織仍在使用當前模式。 要瞭解您的組織使用的模型，請與您的Workfront管理員聯繫。 您可以在此文章的[訪問要求表](#the-access-requirements-table)部分中查找詳細資訊和指向資訊的連結。

>[!NOTE]
>
>如果您對此表中的任何欄位如何應用您有疑問，請與您的WorkfrontFusion管理員聯繫。
